# 📘 HTML Quiz App

A simple and responsive multiple-choice Quiz Application built using **HTML, CSS, and JavaScript**.  
It allows users to answer questions, get instant feedback, and see their final score at the end.

## 🧠 How It Works

1. Questions are stored in a JavaScript array
2. Each question contains multiple answers
3. When an answer is clicked:
   - Correct answer is highlighted in green
   - Wrong answer is highlighted in red
   - All buttons get disabled
4. Clicking **Next** loads the next question
5. Final score is shown at the end
6. Option to restart the quiz appears
